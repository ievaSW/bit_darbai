Kintamieji

Number

Skaičiai.

Stringai

Tai yra tekstiniai kintamieji.

Boolean

Kintamieji, kurie turi reikšmę TRUE arba FALSE

Objektai

Masyvas (let Arry = [ ];) yra vienas iš objektų tipų.

Veiksmai su kintamaisiais

x = x % 3; dalyba is trijų, nusako liekana
console.log(x);
kitas būdas kaip aprašyti aritmetinę operaciją
x += 5;
console.log(x) išvedimas į consolę, taip galima pasitikrinti dalykus su visais kintamaisiais.
Su skaičiais ir net tik atliekami veiksmai + * / -

Jei prie skaičiaus pridedamas tekstas, kintamojo tipas tampa Stringu. Tekstas yra indeksuojamas, iš jo galime pasiimti raides pagal jų vietą tekste. Teksto dalyba iš skaičiaus yra negalima, todėl reikšmė yra NaN (Not a number) Pasinaudoję typeof dalyku :D patikriname kintamojo tipą. Kai kintąjam reikšmė nebūna priskiriama, jo reikšmė būna Undefined. Stringo ilgis gaunamas pasinaudojant savybe kintamasis.length (length yra kintamojo savybė, nurodanti kintamojo ilgį).

  • \t\ tabuliacinis simbolis TAB
  • \n - new line symbol
  • `${imone1}\n${imone2}`
  • parseint funkcija gauna tik sveikuosius skaičius
    skaicius = parseInt(skaiciusTekstas); console.log(skaicius);
  • Math.round apvalina reikšmę
  • Math.floor apvalina reikšmę į mažąją pusę
  • Math.ceil apvalina reikšmę į ceil pusę
  • skaicius.toFixed(2) skliausteliuose nurodomas skaičius atitinka skaičių po kablelio
  • skaicius.toString() kintamasis any
  • String(skaicius) kintamsis var
